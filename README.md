Deskripsi
REVERSE merupakan tools berbasis terminal yang berisi beberapa fitur untuk melakukan encode dan decode pada berbagai format obfuscation Python. Tools ini ditujukan sebagai media pembelajaran reverse engineering dan deobfuscation.
Attention: Only For Reverse!
Tampilan Menu
1. Encode Dengan Format ► Japanese
2. Decode Follow Malware Kid dengan format ► __FOLLOW__MALWARE__
3. Decode Base64 Diviculty ► Easy
4. Decode 3 Layer Hard ► Marshal, ZLib, Base64
5. Decode Format Japanese Medium ► Junk Code With Japanaese
6. Encode Format Deobfuscation Hard ► Hard If You Wnat To Decode
7. Exit
Penjelasan Menu
1. Encode Dengan Format Japanese
Mengubah source code Python menjadi bentuk obfuscation menggunakan karakter atau format bertema Japanese.
Fungsi
Menyulitkan pembacaan source code.
Digunakan sebagai metode obfuscation.
2. Decode Follow Malware Kid
Melakukan decoding terhadap source code yang menggunakan format obfuscation:
__FOLLOW__MALWARE__
Fungsi
Mengembalikan source code ke bentuk yang lebih mudah dibaca.
Membantu analisis script yang menggunakan format tersebut.
3. Decode Base64 Difficulty (Easy)
Melakukan decode terhadap script yang hanya menggunakan Base64 sederhana.
Biasanya menangani:
Base64 satu layer
Base64 bertingkat ringan
